# Admin Font Size Plugin

A simple plugin to adjust the font size in the Grav Admin Panel, supporting both Admin Classic and Admin 2.

## Plugin Settings

| Setting | Default | Description |
|---------|---------|-------------|
| Plugin Status | Enabled | Enable or disable the plugin |
| Admin Font Size | Slightly Larger | Font size scale for the Admin Panel body, form elements, toolbar, and Editor Pro. Options: Default (no changes), Slightly Larger, Moderately Larger, Much Larger |

## Installation

Install via the Grav Admin Panel (Plugins → Add) or via GPM:

```bash
bin/gpm install admin-font-size
```
